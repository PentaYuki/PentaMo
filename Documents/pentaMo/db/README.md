# 🗄️ Database & Models

Quản lý cấu trúc dữ liệu và kết nối cơ sở dữ liệu.

- **`models.py`**: Định nghĩa tất cả các bảng (Tables) như Users, Listings, Conversations, Appointments.
- **`pentamo.db`**: File SQLite chứa dữ liệu thực tế (trong môi trường dev).
- **`postgres/`**: Các script/cấu hình dành cho môi trường production dùng PostgreSQL.
