# 📜 Utility Scripts

Các công cụ dòng lệnh (CLI) để quản trị và bảo trì hệ thống.

- **`run.sh`**: Script khởi động toàn bộ Backend.
- **`system_reset.py`**: Reset toàn bộ dữ liệu (Database, FAISS, Uploads).
- **`import_chat_history.py`**: Nạp dữ liệu từ JSONL và chạy thử nghiệm AI Agent.
- **`sync_feedback_to_faiss.py`**: Đồng bộ phản hồi người dùng vào bộ nhớ AI.
- **`seed_classifier.py`**: Nạp mẫu phân loại ý định cho AI.
