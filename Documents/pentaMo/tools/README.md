# 🔧 AI Skills & Tools

Các công cụ mà AI Agent có thể sử dụng khách quan để tương tác với hệ thống.

- **`handlers_v2.py`**: Chứa các hàm xử lý chính:
  - `search_listings`: Tìm kiếm xe thực tế trong DB.
  - `parse_user_intent_for_search`: Bóc tách thông tin từ câu chat.
  - `book_appointment`: Đặt lịch hẹn xem xe.
  - `detect_risks`: Phát hiện rủi ro lừa đảo hoặc giấy tờ.
